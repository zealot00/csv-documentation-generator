# EU Annex 11 快速参考 / EU Annex 11 Quick Reference

## 什么是 Annex 11? / What is Annex 11?

EU Annex 11 是欧盟药品生产质量管理规范 (EU GMP) 的附录，专门针对计算机化系统。

EU Annex 11 is an annex to EU GMP (Good Manufacturing Practice) specifically for computerized systems.

## 适用范围 / Scope

适用于在 GMP 环境下使用的所有计算机化系统，包括:

- 生产系统
- 实验室系统
- 仓库系统
- 质量管理系统

## 核心原则 / Core Principles

1. **风险管理** - 基于风险的方法
2. **生命周期** - 整个系统生命周期管理
3. **文件化** - 充分文件化
4. **验证** - 系统必须验证

## 主要要求 / Main Requirements

### 1. 验证 / Validation

| 要求 | 描述 |
|------|------|
| 生命周期 | 验证贯穿整个生命周期 |
| 文档化 | 验证活动必须文件化 |
| 风险基础 | 基于风险确定验证范围 |

### 2. 系统安全 / System Security

| 要求 | 描述 |
|------|------|
| 访问控制 | 限制系统访问 |
| 权限管理 | 基于角色的权限 |
| 审计追踪 | 记录所有操作 |

### 3. 数据管理 / Data Management

| 要求 | 描述 |
|------|------|
| 数据完整性 | 确保数据准确完整 |
| 数据备份 | 定期备份 |
| 数据归档 | 长期保存 |

### 4. 变更控制 / Change Control

| 要求 | 描述 |
|------|------|
| 变更申请 | 所有变更需要申请 |
| 影响评估 | 评估变更影响 |
| 审批流程 | 变更需要审批 |
